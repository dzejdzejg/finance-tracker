# 💰 Finance Tracker

A personal finance tracking web application built with **Vanilla JavaScript** — no frameworks, no dependencies (except Chart.js for data visualization).

## ✨ Features

- **Demo mode** — explore the app with pre-loaded sample data
- **User mode** — add and manage your own transactions, budgets, and reminders
- **Dashboard** — income, expenses, and balance overview with trend indicators
- **Transaction management** — add, edit, delete, and search transactions with filters (type, category, date range)
- **Budgets** — set spending limits per category and period (week/month/year) with progress tracking and overspend warnings
- **Reminders** — add reminders with optional due dates, overdue highlighting included
- **Analytics** — interactive charts with range and category filters (expenses by category, spending over time, cashflow, balance growth)
- **Crypto widget** — live crypto prices (BTC, ETH, BNB, SOL, XRP) via CoinGecko API with 15-minute cache
- **Export** — download your data as JSON (full backup) or CSV (transactions for Excel)
- **Dark mode** — full dark theme support
- **Responsive** — mobile-first design with bottom navigation, works on all screen sizes
- **Persistent storage** — data saved to localStorage

## 🛠️ Tech Stack

- **HTML5** — semantic markup, accessibility attributes
- **CSS3** — BEM methodology, CSS custom properties, responsive grid/flexbox layouts
- **JavaScript (ES6+)** — vanilla JS, ES modules, no frameworks
- **Chart.js** — data visualization
- **Font Awesome 7** — icons

## 🚀 Running Locally

```bash
git clone https://github.com/dzejdzejg/finance-tracker.git
cd finance-tracker
```

Open `index.html` with a local server (e.g. [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code). The app uses ES modules and won't work when opened directly as a file.
